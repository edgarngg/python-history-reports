PyhtonReports
=====================================

Entorno Python creado para generar reportes.

Para poder crear un reporte es necesario colocar en la consola:

```sh
$ python report.py --infile audits.json
```

En donde `audits.json` es el archivo que contiene la informaci√≥n necesaria para crear su respectivo reporte.

Para Windows es necesario colocar la ruta donde se encuentra el archivo `report.py`, el archivo json y donde se encuentra Python en el sistema.
